Skola

- Meteor, Blaze, MongoDB, Heroku


This is a side project that I have been working for about 2 months. The main idea here is to create a platform that connects those who are looking for scholarship information and those who know any scholarship information. Ultimately, this application aims to be a community for scholarship seekers and providers. The spark for me to start this project was when I lost my scholarship a few months ago. It made me realize how valuable a scholarship is and we should not take it for granted.
In this project, I explore the awesomeness of Meteor even further. If you have not tried Meteor, I strongly suggest you do so (you will notice in this site I praise Meteor a lot - because it is super cool!). What makes Meteor really great is the presence of ever growing list of useful packages. In this project, one of the most useful packages is Meteor Toys. With the package, you can monitor the state of your database, Session variables. This is really useful as you can not only see but also update the content of the database. As a result, you can easily e.g. toggle true/false a variable in your MongoDB to test how your Blaze function reacts with the change.

Software + Hardware, Awesome!

- Arduino, Raspberry Pi, C


The rite of passage for all computer engineering students - CG3002! This modules challenges us both in software and hardware realm. The aim is to build a wearable device for visually impaired users. When obstacles detected from the sensors, the device will alert the users and suggest the best course of action to stay on track.

First Desktop App!

- Java, Eclipse


In this module, we are given a task to create a to-do manager for desktop. The main challenge is that users should only use keyboard for input. No mouse! This is rather different from what we usually encounter in many apps today. After building this app, we realize how powerful it is to use only keyboard because every time you reach your mouse, it takes a split second and accumulatively, distract the flow of your app usage. That's why iDO++ is simple.powerful.fast!
Eclipse is the integrated development environment (IDE) for this project. It is powerful and I personally like the drag and drop UI builder. It is similar to the one you have in Microsoft's Visual Studio where you can edit the properties of particular elements such as color, onclick events, etc.

Quickly Fix CSS from Your Browser

Configuring the CSS often can be tricky, especially when we take over project from someone else. Generally, we should know how CSS works in affecting how each element is rendered. There might be more than one CSS rule applied to the same element and you may ask, which rule affects the element? As a rule of thumb, CSS rule that is placed last is the one that affects the element, as the name suggests (cascading).

Create Free Personal Website with Github

Recently I went for several interviews to get my first internship. One interesting observation that I had was most of the interviewers asked me for 'personal website/online portfolio'. At that time my expectation was that I only need to bring my hard copy CV or resume and I will be good to go, right? No. Apparently it is the trend now (or perhaps it is just me who were not aware) where your interviewer ask about your personal projects, online portfolio or personal website. If you think about it, online portfolio has many more advantages than the traditional (boring) printed CV. It is more dynamic, more personalized, you can insert photos, videos, sample codes. Not mentioning it is more environmentally friendly. More importantly, you can write articles to share your experience or knowledge about things that you learnt. In other words, you can help others who are facing similar difficulties. Awesome!
